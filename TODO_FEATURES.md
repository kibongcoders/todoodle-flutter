# todoodle 추가 기능 목록

## 우선순위 높음 (High Priority)

### 1. 사전 알림 (Reminder Before Due) ✅ 완료
- **설명**: 마감 전 미리 알림 (10분/30분/1시간/1일 전)
- **예시**: "30분 후 '보고서 제출' 마감입니다"
- **난이도**: 중
- **필요 작업**:
  - [x] Todo 모델에 `reminderOffsets` 필드 추가
  - [x] todo_form_screen에 사전 알림 시간 선택 UI 추가
  - [x] NotificationService에 다중 알림 스케줄링 로직 추가

### 2. 오늘의 할일 / 예정됨 뷰 (Today / Upcoming View) ✅ 완료
- **설명**: 오늘/이번 주/다음 7일 마감 할일 모아보기
- **예시**: 홈 화면에 "오늘", "예정됨" 탭 추가
- **난이도**: 중
- **필요 작업**:
  - [x] HomeScreen에 필터 탭 추가 (전체/오늘/예정됨)
  - [x] TodoProvider에 날짜별 필터링 메서드 추가
  - [x] 빈 상태 UI 개선 (오늘 할일 없음 등)

---

## 우선순위 중간 (Medium Priority)

### 3. 습관 트래커 (Habit Tracker) ✅ 완료
- **설명**: 매일 반복 할일의 연속 달성 일수, 캘린더 히트맵 표시
- **예시**: GitHub 잔디처럼 달성 현황 시각화
- **난이도**: 상
- **필요 작업**:
  - [x] Todo 모델에 `completionHistory` 필드 추가 (완료 날짜 기록)
  - [x] 연속 달성 일수 계산 로직 추가 (`currentStreak`, `longestStreak`)
  - [x] 캘린더 히트맵 위젯 구현 (`HabitHeatmap`, `OverallHabitHeatmap`)
  - [x] 습관 통계 화면 추가 (`HabitScreen`)
  - [x] 하단 네비게이션에 습관 탭 추가

### 4. 알림 시간대 설정 (Do Not Disturb) ✅ 완료
- **설명**: 방해금지 시간 설정 (예: 밤 10시~아침 8시)
- **예시**: 해당 시간에는 알림 자동 무음
- **난이도**: 중
- **필요 작업**:
  - [x] SettingsProvider에 방해금지 시간 저장 (`dndEnabled`, `dndStartTime`, `dndEndTime`)
  - [x] 설정 화면에 시간 범위 선택 UI 추가
  - [x] NotificationService에서 방해금지 시간 체크 로직 추가 (`_adjustForDnd`)

### 5. 스누즈 기능 (Snooze)
- **설명**: 알림을 5분/15분/1시간 후로 미루기
- **예시**: 알림 클릭 시 스누즈 옵션 제공
- **난이도**: 상
- **필요 작업**:
  - [ ] 알림 액션 버튼 추가 (스누즈 옵션)
  - [ ] 알림 클릭 핸들링 구현
  - [ ] 스누즈 알림 재스케줄링 로직

## 우선순위 낮음 (Low Priority)

### 6. 포모도로 타이머 (Pomodoro Timer)
- **설명**: 25분 집중 + 5분 휴식 타이머
- **예시**: 할일별 소요 시간 추적
- **난이도**: 상
- **필요 작업**:
  - [ ] 포모도로 타이머 화면 추가
  - [ ] 타이머 서비스 구현 (백그라운드 동작)
  - [ ] Todo 모델에 `timeSpent` 필드 추가
  - [ ] 통계 화면에 시간 추적 현황 표시

### 7. 위젯 / 퀵액션 (Widget / Quick Action)
- **설명**: 홈 화면 위젯, 앱 아이콘 롱프레스로 빠른 추가
- **예시**: iOS/Android 홈 화면 위젯
- **난이도**: 상
- **필요 작업**:
  - [ ] iOS WidgetKit 구현
  - [ ] Android App Widget 구현
  - [ ] macOS 위젯 구현
  - [ ] Quick Actions (3D Touch / Long Press) 구현

### 8. 위치 기반 알림 (Location-Based Reminder)
- **설명**: 특정 장소 도착/출발 시 알림
- **예시**: "집에 도착하면 '분리수거' 알림"
- **난이도**: 최상
- **필요 작업**:
  - [ ] 위치 권한 및 geolocator 패키지 추가
  - [ ] 장소 저장 기능 (자주 가는 곳)
  - [ ] Geofencing 구현
  - [ ] Todo에 위치 트리거 연결

---

## 구현 순서 제안

1. **1단계**: 사전 알림 + 오늘의 할일 뷰 (기본 기능 강화)
2. **2단계**: 습관 트래커 + 알림 시간대 설정 (사용자 경험 개선)
3. **3단계**: 스누즈 + 포모도로 (고급 기능)
4. **4단계**: 위젯 + 위치 기반 알림 (플랫폼 네이티브 기능)

---

## 참고 사항

- 각 기능 구현 시 Hive 마이그레이션 필요 여부 확인
- 새 필드 추가 시 기본값 설정 필수 (기존 데이터 호환)
- 알림 관련 기능은 플랫폼별 테스트 필수 (macOS, iOS, Android)
