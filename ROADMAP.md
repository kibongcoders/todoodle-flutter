# Todoodle 로드맵 - 100만 사용자를 위한 Doodle 여정 🎨

## 비전

> **"개인 낙서장처럼 편안하고, 스케치북이 채워지듯 즐거운 할일 관리"**

Todoodle은 생산성 앱이 아닌 **성장 일기**입니다.
매일 체크하는 할일이 귀여운 낙서로 그려지고, 손그림 뱃지가 쌓이며, 나만의 스케치북이 완성됩니다.

**구현 예시**:
```dart
// 경험치 계산
int calculateXP(Priority priority) {
  switch (priority) {
    case Priority.veryHigh: return 50;
    case Priority.high: return 30;
    case Priority.medium: return 20;
    case Priority.low: return 10;
    case Priority.veryLow: return 5;
  }
}

// 레벨 테이블
int getLevelFromXP(int totalXP) {
  return (sqrt(totalXP / 100)).floor() + 1; // 레벨당 XP 증가
}
```

## 제외된 기능 (Doodle 정체성 보호) 🚫

다음 기능은 **개인 낙서장 컨셉**과 맞지 않아 로드맵에서 **의도적으로 제외**합니다:

| 기능 | 제외 이유 |
|------|-----------|
| ❌ **AI 기능** (Gemini API) | "스마트 어시스턴트" 느낌 → 장난스러운 Doodle 분위기와 충돌 |
| ❌ **팀 협업** (워크스페이스, 할일 할당) | 기업용 Slack/Trello 느낌 → 개인 노트북 친밀함 상실 |
| ❌ **프리미엄 구독** (유료 기능) | "자유로운 낙서" 정신 위배 → 모든 기능 무료 제공 |
| ❌ **Firebase 클라우드** (인증, Firestore) | 백엔드 복잡도 증가 → 로컬 Hive로 충분 |
| ❌ **FCM 푸시** (서버 알림) | 앱 종료 상태 알림 불필요 → 로컬 알림 유지 |

### 💭 철학

> **"Doodle은 자유롭고, 개인적이며, 장벽이 없어야 한다"**

- 서버 없이 동작하는 순수 로컬 앱
- 로그인 불필요, 구독 불필요, 광고 없음
- 나만의 손그림 노트북, 누구와도 비교하지 않음
- 기능보다 감성으로 사용자와 소통
